# IPL-Data-Analysis
Exploratory Data Analysis of IPL matches (2008-2020) using Python, Pandas, Matplotlib and Seaborn
#  IPL Data Analysis (2008-2020)

##  Project Overview
Exploratory Data Analysis of Indian Premier League (IPL) 
matches from 2008 to 2020 using Python, Pandas, 
Matplotlib and Seaborn.

## Key Insights
-  Most successful team: Mumbai Indians (144 wins)
-  Toss winners win the match ~51% of the time
-  Analyzed 1095 matches and 260,920 ball-by-ball records

##  Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

##  Project Structure
- `ipl_analysis.ipynb` - Main analysis notebook
- `matches.csv` - Match level data
- `deliveries.csv` - Ball by ball data

##  Visualizations
- Top 10 most successful teams
- Toss vs match win analysis
- Top 10 run scorers of all time
- Top 10 wicket takers of all time
- Season wise runs trend
